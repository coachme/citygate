Overview

You can change the way CanCan handles an access denied to the user by writing a rescue_from method in your application_controller.rb

Examples:

CanCan::AccessDenied handling 

rescue_from CanCan::AccessDenied do |exception|
  redirect_to my_custom_path, :alert => exception.message
end
